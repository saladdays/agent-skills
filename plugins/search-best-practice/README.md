# search-best-practice

世の中のベストプラクティスを Web 検索で調査し、鮮度・信頼性・多角的視点を考慮して構造化レポートにまとめるスキルです。

## 特徴

- **Phase 0 メタ判断**: 鮮度・深度・言語・視点・リスクを調査前に自動決定
- **二層ラベルモデル**: 原則（安定）とツール（流動）で鮮度を分離管理
- **反証検索の必須化**: アンチパターンを意図的に探し、一面的な結論を防止
- **2軸リスク判定**: セキュリティリスクと影響リスク（YMYL）を独立評価
- **原理ベース**: キーワードリストではなく原理で判断し、未知のドメインにも対応

## 対応ドメイン

技術・ビジネス・教育・医療・法務・デザイン・クリエイティブ等、あらゆるドメインに対応。
15 トピック × 3 ラウンドのテストで全件完走を確認済み。

## 使い方

```
/search-best-practice Kubernetes シークレット管理のベストプラクティス
/search-best-practice SaaSの価格設定のベストプラクティス
/search-best-practice 子どもの読書習慣の身につけ方
```
